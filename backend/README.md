
# Waves - Chat App  

## Backend  

Construido en Laravel, provee una API REST para administrar:  

- Usuarios
- Amigos
- Chats
- Mensajes
- Archivos adjuntos
- _**Waves**_

## POSTMAN

Esta API puede probarse por medio de POSTMAN, con las colecciones y entornos de prueba en la carpeta /postman
