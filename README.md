
# Waves - Chat App

Una aplicación de chat que presenta Waves, una divertida forma de mostrar tus pensamientos, actividades, planes, humor. ¡Tu propio espacio personalizable!.


Objetivo: Aprender y demostrar una arquitectura limpia de diseño tanto de estructura de datos como UX/UI.

Estado: En desarrollo.
Stack principal: Laravel (Rest API) | React.JS (Frontend SPA).
